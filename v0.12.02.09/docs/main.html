WEBAPP


The goal of the WebApp Class is to harmonize working ways of each developers.


It is working with the MVC (model,view,controller) model.


Each call to the website is done via a parameter in the url called « component ». This means the name of the « controller » .


ex : http://www.website.com/index.php?component=hello&action=world


When the controller is loaded, he create the « model » and the « view ».


The “model” is where you implement the access to the data (database, XML files, web services, etc...)


The “View” is where you set the result of the model in a Visual result (HTML, XML, JSON, etc...).





Main Process :


  • WEBAPP create


      • Load “main.conf.php” file.

      • Load “main.php” langage file (languages Dir).

      • Set the request parameters in the property “UrlVars”.

      • Set component and action names in properties. ($this->component, $this->action).

      • Call the checkBrowser method if it is configured.

  • WEBAPP method “execute” called.


      • Check locally (website component Dir) if a component exist. If not, check it globally (WEBAPP “component” Dir). If not , the default one set in configuration file is used.

      • LoadMainMenu method called.

      • LoadComponent method called.

      • LoadActionMenu method called.

      • LoadModules method.


  • WEBAPP method “loadtemplate” called.


      • Switch on ModeRespone (template,rest (XML),onlybody)

      • Template case

        • Load/add component/action “js” code found in template “js” directory in $this->jsCode propertie.

        • Load/add component/action “js” code found in site “js” directory in $this->head propertie as a Filelink ('<script type="text/javascript" src="' . $cSiteJsFile . '"></script>') .

        • Load/add component/action “CSS” code found in template “CSS” directory in $this->cssCode propertie.

        • Load/add component/action “CSS” code found in site “CSS” directory in $this->cssCode propertie.

        • Set in $this->head propertie links to css and js files found arrays $this->jsFiles and $this->cssFiles

        • If file exists, include template index file.
